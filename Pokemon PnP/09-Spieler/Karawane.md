## Freizeitaktivitäten

| Aktivität                           | Effekte/Boni/Notizen          |
| ----------------------------------- | ----------------------------- |
| [[Attacken beibringen]]             | Bis APK 1 ohne kosten         |
| [[Gegenstände herstellen - suchen]] | [[Sinelbeeren]] im Wald - 2W4 |
| [[Zuneigung steigern]]              | bis ZW 50, <br>+1 Bonus       |
| [[Statuswerte trainieren]]          | /                             |

### [[Will]]
hat euch die Basics der [[Pokémon]] Pflege gezeigt. Ihr könnt nun:
- Attacken mit AP-Kosten von 1 Problemlos beibringen
- Statuswerte Trainieren
- Zuneigung steigern bis 50
- Beeren suchen (2W4 [[Sinelbeeren]]) 
	- solange Ihr euch im Wald befindet

### Tanztruppe
[[Myrapla]], [[Knofensa]] und [[Hopsproß]]
- +1 auf Zuneigungsgewinn bei der Freizeitaktivität [[Zuneigung aufbauen]]
