# Trainer-Level

Genau wie Pokémon haben Trainer auch ein Level.

Immer wenn du eine **Spielrunde abschließt**, erhält dein Trainer einen **Erfahrungspunkt** (EP).

Um ein Level aufzusteigen brauchst du:

```
Benötigte EP = 3
```

Bei einem Level-Up erhältst du:

- **1 Talentpunkt** → Siehe [[Talente]]
- **3 Fertigkeitspunkte** → Siehe [[Fertigkeiten]]
- **3 Wissenspunkte** → Siehe [[Wissen]]
- **Aspektmodifikation** → Passe einen deiner Aspekte an, erhalte einen neuen, lege einen alten ab. Kläre deine Entscheidung mit dem SL ab.

Du kannst diese Punkte direkt einsetzen um [[Fertigkeiten]] oder [[Wissen]] zu verbessern. Unbenutzte Punkte werden dem jeweiligen Punktekonto gutgeschrieben.

| Aktueller Wert | Kosten pro +1 |
| -------------- | ------------- |
| ≤ 3            | 1 Punkt       |
| 4-5            | 1 Punkt       |
| 6-7            | 2 Punkte      |
| 8-9            | 3 Punkte      |
**Maximum:** 10
