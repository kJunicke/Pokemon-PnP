[[Attacken]]

Eine Serie an Kratzenden Angriffen

| [[Typ]]                 | normal   |
| ----------------------- | -------- |
| [[Kategorie]]           | physisch |
| [[Genauigkeit]]         | -5       |
| [[Reichweite]]          | nah      |
| [[Ausdauer\|AP-Kosten]] | 1        |
| Schadenswürfel          | 1W4 x 5  |
#### Effekt
- Wirf 5 W20, als wären es einzelne Angriffe
- Jeder Treffer erhöht den Schaden der Attacke um 1W4
- Addiere deinen Angriff nur auf den Gesamtschaden (nicht jeden einzelnen Treffer)
